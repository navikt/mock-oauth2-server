withExtraMatchParams

fun withExtraMatchParams(extraMatchParams: Map<String, String>): OAuth2TokenCallback

Returns a view of this callback that supplements matching with extraMatchParams — key/value pairs that are considered when no matching form parameter is found in the token request body.

Intended for use during interactive login, where the login username is injected under the SUBJECT_PARAM key so that a RequestMapping with requestParam = "subject" can match on it:

{ "requestParam": "subject", "match": "alice", "claims": { "role": "admin" } }
